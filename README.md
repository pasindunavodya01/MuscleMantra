# 🏋️ Gym Member Management System

A full-stack system for managing gym members, attendance, and payments.

## ✨ Features
- Admin-based member registration
- QR code check-in / check-out system
- Attendance tracking
- BMI tracking
- Payment tracking
- WhatsApp notifications for reminders
- Image upload with Cloudinary

## 🛠 Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Auth: Firebase Authentication
- Media: Cloudinary

## 🚀 How It Works
- Admin registers members
- Members check-in via QR code
- System tracks attendance and payments
- Notifications sent via WhatsApp

## 📦 Installation
```bash
git clone <repo-url>
npm install
npm run dev
